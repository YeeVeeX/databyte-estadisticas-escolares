Funciones de Lógica

def obtener_edad_mayor(alumnos):
    mayor = -1
    nombre_mayor = ""
    for nombre in alumnos:
        if alumnos[nombre] > mayor:
            mayor = alumnos[nombre]
            nombre_mayor = nombre
    return nombre_mayor, mayor

def calcular_promedio(alumnos):
    suma = 0
    cantidad = 0
    for nombre in alumnos:
        suma += alumnos[nombre]
        cantidad += 1
    return round(suma / cantidad, 1)
Sin built-ins: implementamos max, sum y len manualmente para demostrar el algoritmo. El return múltiple devuelve nombre y edad juntos.
round(promedio, 1) — el cliente pidió exactamente un decimal.

Cálculo de la Mediana

def calcular_mediana(alumnos):
    edades = []
    for nombre in alumnos:
        edades.append(alumnos[nombre])
    edades.sort()

    cantidad = len(edades)
    mitad = cantidad // 2

    if cantidad % 2 == 1:
        return edades[mitad]             # Impar
    else:
        return (edades[mitad-1] + edades[mitad]) / 2  # Par

¿Por qué HTML?

  • Cero librerías — reportlab/fpdf requieren pip
  • CSS profesional — gradientes, cards, tablas estilizadas
  • Ctrl+P → PDF desde cualquier navegador
  • Universal — cualquier sistema lo abre
Cada reporte se guarda con timestamp único — nunca se sobrescriben. También se genera un TXT como respaldo.
